# GeoModeler3D
A 3d geometry shape renderer for diagnostics of data like point-cloud, triangles, circles

# Set up
 
## mermaid diagram
Class diagram and other UML diagrams are created in mermaid.
https://mermaid.live/

Option 1: Install the extension in VS Code.
Option 2: Use online editor
Option 3: CLI (generate an image)
•	Install mermaid CLI: npm i -g @mermaid-js/mermaid-cli
•	Render to SVG/PNG: mmdc -i docs/high_level_architecture.mermaid -o docs/high_level_architecture.svg
